📊 Sales Dashboard using Power BI
📌 Project Overview

This project demonstrates an interactive Sales Dashboard created in Microsoft Power BI to analyze sales data and provide actionable business insights.

The dashboard enables users to monitor KPIs, explore sales trends, analyze product categories, compare payment methods, and filter data dynamically.

🚀 Features
Total Sales KPI
Total Orders KPI
Total Quantity Sold
Average Order Value
Monthly Sales Trend
Category-wise Sales Analysis
Category-wise Quantity Sold
Payment Method Analysis
Discount Analysis
Interactive Slicers

🛠️ Tools Used
Microsoft Power BI
Power Query
DAX
Data Cleaning
Data Visualization

📈 Insights
Clothing is the highest-selling category.
Sales fluctuate across different months.
Payment methods contribute almost equally to total sales.
Most orders are placed without discounts.
Interactive filters allow detailed business analysis.
